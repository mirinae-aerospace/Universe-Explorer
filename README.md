# Universe-Explorer
Program for exploring the universe

##About This Program
Universe Explorer is a program for exploring the universe, written in Python Tkinter.
It provides access to and visualization of astronomical data, including SIMBAD and SDSS.
It offers celestial object searches, and 3D space map.
This program allows you to find information on virtually every celestial object outside our solar system, and visualize the universe in 3D.

Universe Explorer는 Python Tkinter로 제작된 우주를 탐험해 볼 수 있는 프로그램입니다. 
SIMBAD와 SDSS 등 천문 데이터에 접근하고 시각적으로 보여줄 수 있습니다.
천체 검색 및 3D 우주 지도 등을 할 수 있습니다.
이 프로그램을 통해 태양계 밖 거의 모든 천체의 정보를 찾을 수 있고, 우주를 3차원으로 시각화할 수 있습니다.
